# Blog Images

Place your converted WebP images in this folder.

## Required Images & Names:

1. **calories-guide.webp** - "What Are Calories" article
2. **macros-guide.webp** - "What Are Macros" article
3. **tdee-calculator.webp** - "How to Calculate TDEE" article
4. **nutribullet-pro-900.webp** - NutriBullet blender review
5. **macro-calculator.webp** - Macro Calculator Guide
6. **core-fitness-dumbbells.webp** - Core Fitness dumbbells (beginner/intermediate)
7. **nuobell-dumbbells.webp** - Nuobell dumbbells (advanced)

## Image Specs:
- **Dimensions:** 1600x900px (16:9 ratio)
- **Format:** WebP
- **File size:** Under 200KB recommended
- **Quality:** 75-85 in Squoosh

## Fallback:
If an image is missing, the blog will automatically show the emoji placeholder instead.
