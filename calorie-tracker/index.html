How Free Calorie Track Works

Set up your profile with age, sex, height, weight, and activity level. Free Calorie Track calculates your BMR (Basal Metabolic Rate) using the clinically-validated Mifflin-St Jeor equation. TDEE (Total Daily Energy Expenditure) is calculated by multiplying BMR by your activity factor. Choose a fitness goal (muscle gain, fat loss, maintenance, athletic performance) for automatic macro targets based on sports nutrition research, or set custom macro goals.

Log food by scanning barcodes, searching the database, selecting from 2,000+ common foods, or creating custom entries. Macros (protein, carbs, fat) are automatically tracked with real-time progress bars. Log exercise from 250+ activities for accurate calorie burn using MET (Metabolic Equivalent) values. Track water intake throughout the day. View NET calories on dashboard showing true energy balance.

Access trends to see weekly and monthly averages for calories and macros. View weight progress charts. Edit any past day without restrictions. Export all data as JSON backup file for safekeeping or device transfers. All data stays on your device for complete privacy.
